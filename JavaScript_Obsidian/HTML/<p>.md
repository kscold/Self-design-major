- `<p>` 태그는 paragraph, 즉 문단의 약자로, 하나의 문단을 만들 때 쓰인다.

## 문법

```html
<p>문단1</p>
```

## 예시

```html
<html>
	<body>
		<p>first paragraph</p>
		<p>second paragraph</p>
		<p>
			new line<br>
			third paragraph
		</p>
	</body>
</html>
```

```bash
// 출력 결과
>> first paragraph
>> second paragraph
>> new line 
>> third paragraph
```
