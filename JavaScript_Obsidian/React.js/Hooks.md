- Hooks는 리액트 v16.8에 새로 도입된 기능이다.

- [[함수형 컴포넌트(Functional Component)]]에서도 상태([[state]]) 관리를 할 수 있는 [[useState()]], 렌더링 직후 작업을 설정하는 [[useEffect()]] 등의 기능을 제공하여 기존의 함수 컴포넌트에서 할 수 없었던 다양한 작업을 할 수 있게 해준다.


## Hooks 종류

- [[useRef()]]
- [[useMemo()]]
- [[useState()]]
- [[useEffect()]]
- [[useReducer()]]
- [[useReducer()]]
