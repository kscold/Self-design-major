- 레코드(Record)는 논리적으로 연관된 [[열(Colum)]]의 집합을 의미한다.
- 엑셀의 행(row)에 해당된다. 
- 데이터를 의미한다.
- 튜플 Tuple이라고 불리기도 한다.

- 여기서 각각의 필드는 특정한 데이터 타입과 크기가 지정되어 있다.
- 즉, 여러 행이 모여 한 열(column)을 이루듯이 여러 [[Spring/SQL/필드(Field)|필드(Field)]]가 모여 한 레코드를 이루는 것이다.
- 따라서 테이블에 행 단위로 저장된 하나하나를 레코드라고 한다.
